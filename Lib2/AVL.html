ダウンロード

概要

AVL木です。

__init__(li=[])

liを元にAVL木を構築します。\( \mathrm{O(len(li) log len(li))} \)かかります

add(x)

xを追加します。追加出来た時はTrue、既に追加済みの時はFalseを返します。
\( \mathrm{O(logN)} \)です。

__getitem__(index)

tree[index]などで小さい方からindex番目(0-indexed)の要素を取得します。
indexが負の値か、len(tree)以上の場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

getindex(value)

このAVL木におけるvalueのインデックスを返します(0-indexed)。
もしvalueが含まれていない場合は挿入されるべきインデックスをxとすると、~x(つまり、-(x+1))を返します。
すなわち、valueが含まれていない場合は負の値が返ってくることが保証されます。
\( \mathrm{O(logN)} \)です。

remove(x)

xをこのAVL木から削除します。このAVL木にxが含まれていた場合はTrue、含まれていなかった場合はFalseを返します。
\( \mathrm{O(logN)} \)です。

first()

このAVL木の最小の要素を返します。このAVL木が空の場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

last()

このAVL木の最大の要素を返します。このAVL木が空の場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

__contains__(x)

x in treeなどで、このAVL木にxが含まれるかを返します。
\( \mathrm{O(logN)} \)です。

poll_first()

このAVL木の最小の要素を返し、削除します。
\( \mathrm{O(logN)} \)です。

poll_last()

このAVL木の最大の要素を返し、削除します。
\( \mathrm{O(logN)} \)です。

ceiling(x)

x以上で最小の要素を返します。もしそのような要素が存在しない場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

higher(x)

xよりも真に大きい最小の要素を返します。もしそのような要素が存在しない場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

floor(x)

x以下で最大の要素を返します。もしそのような要素が存在しない場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

lower(x)

xよりも真に小さい最大の要素を返します。もしそのような要素が存在しない場合はNoneを返します。
\( \mathrm{O(logN)} \)です。

clear()

このAVL木を空にします。
\( \mathrm{O(1)} \)です。

is_empty()

このAVL木が空かを返します。
\( \mathrm{O(1)} \)です。

tolist()

このAVL木と同等のリストを返します。リストの中身は昇順であることが保証されます。
\( \mathrm{O(N)} \)です。

__len__()

このAVL木の要素数を返します。
\( \mathrm{O(1)} \)です。

__del__()

このAVL木を削除します。

__str__()

このAVL木を表す文字列を返します。
このメソッドはstr(tree.tolist())と同等です。
\( \mathrm{O(N)} \)です。

__hash__()

このAVL木を表すハッシュ値を返します。
\( \mathrm{O(1)} \)です。

__setitem__(key, value)

このAVL木にkeyが含まれていたらvalueに変換します。
\( \mathrm{O(logN)} \)です。
コード例
tree[key] = value

__delitem__(key)

このAVL木にkeyが含まれていたら削除します。
\( \mathrm{O(logN)} \)です。
コード例
del tree[key]

その他、setと同様に&、|、^、&=、|=、^=、==、!=演算子に対応しています。
また、+=、-=演算子、iterメソッドにも対応しております。詳しくは実装をご確認ください。

	def first(self):
		if self._root is None:
			return None
		return self.__getfirstnode(self._root)._value

	def __getfirstnode(self,node):
		par = None
		while node is not None:
			par = node
			node = par._left
		return par

	def last(self):
		if self._root is None:
			return None
		return self.__getlastnode(self._root)._value

	def __getlastnode(self,node):
		par = None
		while node is not None:
			par = node
			node = par._right
		return par

	def __contains__(self,x):
		if self._root is None:
			return false
		return __getnode(x) is not None

	def poll_first(self):
		if self._root is None:
			return None
		self._size -= 1
		mini = self.__getfirstnode(self._root)
		ans = mini._value
		self.__delete(mini)
		self._hash ^= hash(ans)
		return ans

	def poll_last(self):
		if self._root is None:
			return None
		self._size -= 1
		maxi = self.__getlastnode(self._root)
		ans = maxi._value
		self.__delete(maxi)
		self._hash ^= hash(ans)
		return ans

	def ceiling(self,x):
		if self._root is None:
			return None
		return self.__ceiling(self._root,x)

	def __ceiling(self,node,x):

			if x>node._value:
				node = node._right
			elif x=node._value:
				node = node._right
			else:
				ans = node
				node = node._left
		return ans._value if ans is not None else None

		rs = 0 if node._right is None else node._right._size
		node._size = ls+rs+1
